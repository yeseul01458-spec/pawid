# PAWID

> 정답이 아닌, 내 반려견만을 위한 기준

## 📌 프로젝트 소개

PAWID는 반려견의 행동을 단순히 기록하는 것을 넘어, 개별 반려견의 성향과 생활 패턴을 학습하여 맞춤형 분석과 상담을 제공하는 AI 기반 반려견 케어 플랫폼입니다.

현재 대부분의 반려견 정보는 일반화된 기준을 제공합니다.

예를 들어 같은 꼬리 흔들기 행동이라도 어떤 강아지에게는 반가움의 표현일 수 있지만, 다른 강아지에게는 긴장 상태의 신호일 수 있습니다.

PAWID는 이러한 문제를 해결하기 위해 반려견 개체별 프로필, 일상 기록, 행동 데이터를 통합 분석하여 보다 개인화된 해석을 제공하는 것을 목표로 합니다.

---

## 🎯 프로젝트 목표

기존 반려동물 플랫폼의 한계

* 일반적인 행동 정보 제공
* 개체별 특성 반영 부족
* 기록 데이터 활용 부족
* 보호자 경험에 의존한 판단

PAWID가 해결하고자 하는 문제

* 반려견별 행동 기준 생성
* 지속적인 행동 패턴 추적
* AI 기반 맞춤형 상담 제공
* 보호자의 의사결정 지원

---

## ✨ 주요 기능

### 1. 반려견 프로필 관리

반려견의 기본 정보를 등록하고 관리할 수 있습니다.

등록 정보

* 이름
* 견종
* 생년월일
* 성별
* 중성화 여부
* 체중
* 알레르기 정보
* 입양 정보
* 생활 환경

이를 통해 모든 분석의 기초 데이터를 구축합니다.

---

### 2. 행동 기록 시스템

하루 단위로 반려견 상태를 기록할 수 있습니다.

기록 항목

* 식사량
* 음수량
* 산책 시간
* 배변 상태
* 수면 상태
* 특이 행동
* 건강 상태

누적 데이터를 통해 장기적인 패턴을 확인할 수 있습니다.

---

### 3. AI 행동 상담

보호자가 궁금한 상황을 자연어로 입력하면 AI가 답변을 제공합니다.

예시

* "우리 강아지가 갑자기 하울링을 해요."
* "산책 중 특정 장소에서만 짖어요."
* "최근 잠자는 시간이 줄어든 것 같아요."

AI는 일반적인 정보뿐 아니라 등록된 반려견 프로필과 기록 데이터를 함께 고려하여 답변합니다.

---

### 4. 맞춤형 분석 시스템

PAWID의 핵심 기능입니다.

단순 행동 해석이 아닌 반려견 개체 기준으로 분석을 수행합니다.

분석 요소

* 행동 빈도 변화
* 생활 패턴 변화
* 건강 관련 이상 징후
* 스트레스 지표
* 사회성 변화

분석 결과는 시각화된 리포트 형태로 제공됩니다.

---

### 5. AI 영상 분석 (개발 진행)

반려견 영상을 분석하여 행동을 자동으로 인식하는 기능입니다.

분석 예정 요소

* 꼬리 움직임
* 귀 방향
* 자세 변화
* 보호자와의 거리
* 시선 방향
* 이동 패턴

동일 행동이라도 개체 특성을 고려한 해석을 목표로 합니다.

---

### 6. 산책 친구 매칭 (확장 기능)

지역 기반 반려견 매칭 서비스입니다.

매칭 기준

* 견종
* 체급
* 나이
* 성향
* 사회성

이를 통해 반려견에게 적합한 산책 친구를 찾을 수 있습니다.
